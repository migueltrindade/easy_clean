INTRODUCTION
------------
Easy Clean is a perfect multi-region base theme for who loves to build up a website from scratch.
It includes an HTML5-friendly CSS Reset, Normalize CSS & other best practices from Paul Irish's HTML5 Boilerplate.


INSTALATION
------------
Extract the theme in your sites/all/themes/ directory and enable it.


USEFUL INFORMATION
------------------
Easy Clean applies classes to the <html> tag to enable easier styling for Internet Explorer

.ie7 .selector { ... }
.ie7 #selector { ... }
.ie7.ie8 #selector { ... }
